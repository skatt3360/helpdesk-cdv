# VinciDesk — CDV IT Helpdesk v1.0

Profesjonalny panel Helpdesku IT **Collegium Da Vinci**.

## Funkcje
- Realtime tickety i zgłoszenia IT
- Zadania wewnętrzne zespołu
- Kalendarz + harmonogram dyżurów i prac konserwacyjnych
- Metryki SLA i KPIs Helpdesku
- Chat zespołu + status online
- Push notifications (Android + iOS 16.4+ PWA)
- Pełna obsługa offline + lokalny backup

## Deploy na Netlify
Wrzuć folder na GitHub → New site w Netlify → import z GitHub.  
netlify.toml jest już skonfigurowany.

## Push na iOS
Dodaj stronę do ekranu głównego → zaloguj się → włącz powiadomienia.

Gotowy do codziennej pracy zespołu IT CDV.
